## Project Overview

This is a **check-in system** (`checkin_system`) project, currently in early development. The working directory is part of a broader development environment that also contains related systems (`attendance-system`, `club-system`).

The project uses Traditional Chinese (繁體中文) as the primary language for UI and documentation.

## Custom Commands

- `/review` — Reviews current code changes (uses `git diff HEAD`). Checks code quality, style consistency, potential bugs, security issues, performance, and test coverage.
