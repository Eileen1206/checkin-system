# 政旭汽車材料行 打卡系統 — 部署說明

系統部署於 [Railway](https://railway.app/)，資料庫使用 Railway 提供的 PostgreSQL。

---

## 一、Railway 服務架構

| 服務名稱 | 類型 | 啟動指令 |
|---|---|---|
| `checkin-system` | Web Service | `gunicorn checkin_system.wsgi` |
| `crontab提醒打卡` | Cron Service | `python manage.py migrate && python manage.py remind_attendance` |

Cron 排程：`*/5 * * * *`（每 5 分鐘）

---

## 二、環境變數設定

在 Railway 的 `checkin-system` 服務 → Variables 填入：

```env
SECRET_KEY=（隨機產生的金鑰）
DEBUG=False
ALLOWED_HOSTS=checkin-system-production-4544.up.railway.app

# LINE Messaging API
LINE_CHANNEL_SECRET=
LINE_CHANNEL_ACCESS_TOKEN=
LINE_BOT_BASIC_ID=
MANAGER_LINE_USER_ID=

# LINE LIFF（LINE Login channel）
LIFF_DELIVERY_ID=

# LINE Rich Menu ID（執行 setup_richmenu 後取得）
RICHMENU_DELIVERY=
RICHMENU_STAFF=

# 公司 GPS 座標
OFFICE_LAT=
OFFICE_LNG=
OFFICE_RADIUS_METERS=100

# 路線規劃 API（OpenRouteService）
ORS_API_KEY=

# RFID
RFID_API_KEY=

# PostgreSQL（Railway 自動注入，通常不需手動填）
DATABASE_URL=
```

---

## 三、LINE Bot 設定

1. 前往 [LINE Developers Console](https://developers.line.biz/)
2. Messaging API channel → **Webhook URL**：
   ```
   https://checkin-system-production-4544.up.railway.app/attendance/webhook/
   ```
3. 開啟 **Use webhook**，關閉 **Auto-reply messages**

---

## 四、LINE LIFF 設定

1. 建立 **LINE Login channel**（與 Messaging API channel 不同）
2. 在 LIFF 分頁新增 LIFF app：
   - Endpoint URL：`https://checkin-system-production-4544.up.railway.app/liff/delivery/`
   - Scope：`profile`
   - Add friend option：`Off`
3. 將 LIFF ID 填入 Railway 環境變數 `LIFF_DELIVERY_ID`
4. 發布 LINE Login channel（否則員工無法使用）

---

## 五、初次部署後

```bash
# 建立 cache 資料表（只需執行一次）
python manage.py createcachetable

# 建立管理員帳號
python manage.py createsuperuser

# 建立 LINE Rich Menu（只需執行一次）
python manage.py setup_richmenu
```

在 Railway 可透過 **Deploy → Run command** 執行上述指令。

---

## 六、Django Admin 群組設定

進入 `https://checkin-system-production-4544.up.railway.app/admin/`

1. **Groups** → 新增 `admin` 群組
2. 新增 `finance` 群組
3. 將對應使用者加入群組

---

## 七、RFID 刷卡機（店內電腦）

店內電腦需持續執行 `rfid_listener.py`：

```bash
pip install keyboard requests plyer
python rfid_listener.py
```

> 需要以**系統管理員身分**執行（keyboard 套件需要）

---

## 八、常見問題

| 問題 | 解決方式 |
|---|---|
| LINE Bot 無回應 | 確認 Webhook URL 正確，Railway 服務正常運行 |
| LIFF 顯示 400 | 確認 LINE Login channel 已發布（非 Developing） |
| LIFF 顯示缺少任務參數 | 必須從 LINE 訊息的按鈕進入，不能直接開網址 |
| 打卡提醒沒收到 | 確認 Cron 服務正常、員工有開啟提醒、今天是工作日 |
| 忘記管理員密碼 | Railway Run command：`python manage.py changepassword 帳號` |
