# 政旭汽車材料行 打卡系統 — 部署說明

## 一、環境需求

| 項目 | 版本 |
|------|------|
| Python | 3.10 以上 |
| pip | 隨 Python 安裝 |
| 作業系統 | Windows 10/11 |

---

## 二、安裝步驟

### 1. 複製專案資料夾

將整個 `checkin_system` 資料夾複製到目標電腦。

> ⚠️ 注意：`.env` 檔案包含敏感資訊，**不要**放進公開的雲端或 USB，請單獨傳遞。

---

### 2. 安裝 Python

前往 https://www.python.org/downloads/ 下載並安裝 Python 3.10 以上版本。

安裝時勾選 **「Add Python to PATH」**。

安裝完成後，開啟命令提示字元（cmd）確認：

```
python --version
```

---

### 3. 安裝相依套件

在命令提示字元切換到專案資料夾：

```
cd 專案路徑\checkin_system
```

安裝套件：

```
python -m pip install -r requirements.txt
```

---

### 4. 建立 `.env` 檔案

在專案根目錄（與 `manage.py` 同層）建立 `.env` 檔案，內容參考 `.env.example`：

```
SECRET_KEY=（填入金鑰）
DEBUG=True

LINE_CHANNEL_SECRET=（填入 LINE Channel Secret）
LINE_CHANNEL_ACCESS_TOKEN=（填入 LINE Channel Access Token）
LINE_BOT_BASIC_ID=（填入 LINE Bot Basic ID，如 @xxx）
MANAGER_LINE_USER_ID=（填入管理員的 LINE User ID）

OFFICE_LAT=（公司緯度）
OFFICE_LNG=（公司經度）
OFFICE_RADIUS_METERS=100
```

> 如果不確定公司座標，可以在 Google Maps 上點擊公司位置，網址列會顯示經緯度。

---

### 5. 建立資料庫

```
python manage.py migrate
```

---

### 6. 建立管理員帳號

```
python manage.py createsuperuser
```

依提示輸入帳號、信箱、密碼。

---

### 7. 建立群組權限

進入後台 http://127.0.0.1:8000/admin/

登入後：
1. 點選 **Groups** → 新增群組 `admin`
2. 再新增群組 `finance`
3. 將對應使用者加入群組

---

### 8. 啟動伺服器

```
python manage.py runserver
```

開啟瀏覽器前往：http://127.0.0.1:8000/dashboard/

---

## 三、每次開機使用

每次要使用系統時，開啟命令提示字元執行：

```
cd 專案路徑\checkin_system
python manage.py runserver
```

然後在瀏覽器開啟 http://127.0.0.1:8000/dashboard/

---

## 四、LINE Bot 設定（如需重新設定）

1. 前往 [LINE Developers Console](https://developers.line.biz/)
2. 選擇你的 Channel → Messaging API
3. Webhook URL 填入：`https://你的網域/attendance/webhook/`
4. 開啟 **Use webhook**
5. 關閉 **Auto-reply messages**

> 本機開發需使用 ngrok 產生公開網址：
> ```
> ngrok http 8000
> ```
> 將 ngrok 產生的 https 網址填入 Webhook URL。

---

## 五、常見問題

| 問題 | 解決方式 |
|------|---------|
| `pip` 無法辨識 | 改用 `python -m pip install` |
| 啟動後看不到頁面 | 確認網址是 http 不是 https |
| LINE Bot 無回應 | 確認 `.env` 的 Token 正確，且伺服器正在執行 |
| 忘記管理員密碼 | `python manage.py changepassword 帳號名稱` |
